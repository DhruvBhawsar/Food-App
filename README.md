<h3>Technologies we used:</h3>
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
<ul/>
<h3>Features:</h3>
<li>API Fetching</li>
<li>Sign-in/Sign-up</li>
<li>Display random recipie</li>
<li>Display search input</li>
